The tech stack for this project is 
Flask and Linux for the backend, 
and HTML, CSS, and Javascript for the
front-end

In addition to the requirements file being run, you must also unzip the files within the .vneu directory

The tree structure for the project is as
follows:
C:.
│   hi.txt
│   ReadMe.txt
│   requirements.txt
│   run.py
│   tree.txt
│
└───app
    │   DBconnect.py
    │   form.py
    │   routes.py
    │   __init__.py
    │
    ├───config
    │       config.json
    │
    ├───logs
    │       appinfo.log
    │
    ├───static
    │       style.css
    │       styleDiagnostic.css
    │       stylePlaylist.css
    │       styleSearch.css
    │
    └───templates
            diagnostic.html
            home.html
            playlist.html
            search.html
